# @anjartiyo/giogia-core

Shared types and interfaces for the GioGia web testing framework.

## Types

- `SemanticElement` - Semantic page element descriptor
- `PageSnapshot` - Full page semantic snapshot
- `LocatorStrategy` - Resolution strategy (semantic-id, role, label, text, test-id, css, xpath)
- `AIProvider` - Supported AI providers
- `DiscoveredFlow` - Workflow from exploratory testing
- `ExploratoryReport` - Full exploration report
- `BDDScenario` - Parsed Gherkin scenario
- `GeneratedTest` - AI-generated test output
- `TestResult` / `TestReport` - Execution results

## Install

```bash
npm install @anjartiyo/giogia-core
```
